Chat Widget Preview

This page loads the same chatbot embed used on the website, so you can test the real widget in one place.

The current setup uses local fallback replies and sends lead saves to Formspree, so it stays usable even without a separate chat API.

  1. Open the floating chat button in the bottom-right corner.
  2. Try quick replies like JDA Approved plots or Free site visit.
  3. Share a phone number in chat and the lead save prompt will appear.
